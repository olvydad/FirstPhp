Ce projet consiste à développer un site web pour un hôtel (Olymax) permettant de présenter
(les services, les chambres et les informations de réservation etc..) aux clients.
L'objectif est de pratiquer les bases du développement web avec HTML en créant une interface
simple et claire pour un site d’hôtel.
**Objectifs du projet**
Créer une structure de page web avec HTML;
Présenter les services de l’hôtel;
Afficher les chambres disponibles;
Ajouter un formulaire de réservation ou de contact;
Organiser correctement les fichiers d’un projet web...
**Fonctionnalités**
Le site comprend :
1) Une page d’accueil
- Une section présentant l’hôtel
- Une section présentant les chambres
- Une section présentant les services
2) Un formulaire de réservation ou de contact
**Technologies utilisées**
Ce projet utilise les technologies suivantes :
- HTML5 – Structure des pages web
- CSS3 – Mise en forme du site
- Git & GitHub – Gestion de version du projet
- js - pour les interactions
